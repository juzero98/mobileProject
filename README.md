# 개요
기존에 쓰는 여러 캘린더 어플의 불편함을 개선하고 장점은 활용한 캘린더 어플


# 주요기능

1. 다른 사람과 일정 공유
2. 일정추가 및 변경 시 피드알림

3.각 일정/ 피드에 댓글추가기능

4.드래그를 통한 여러날짜 한번에 일정추가 

  ex)9박10일 여행시 드래그 한번으로 일정추가 가능

5.일정 카테고리 별 색 선택 가능

6.보다 편리한 메모기능
