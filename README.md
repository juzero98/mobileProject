# 개요
항상 학식당에 주문 줄이 길었고 메뉴선택의 어려움이 있었다. 그리고 피드백을 전달하는 데에 어려움도 있는 것 같아 그 부분을 보완하고자 학식당 어플을 만들고자 한다.

# 주요기능

1. 오늘의 학식 메뉴를 볼 수 있음

2. 어플로 메뉴 선택 및 결제 가능함

3. 주문한 사람만 피드백 가능 (별점과 간단한 리뷰)

4. 인기 메뉴 추천

        ex) 3번에서 얻은 별점을 토대로 인기 메뉴 추천

5. 자주 먹는 메뉴 산출

6. 학식 포인트 충전 가능

7. 학식 이벤트
